# 🐍 Snake Arena

A simple and fun Snake Game built using React.js with smooth gameplay and a modern UI.

## 🎮 Features

- Smooth snake movement
- Score tracking
- Restart game option
- Responsive design

## 🛠️ Technologies Used

- React.js
- JavaScript
- CSS

## ⌨️ Controls

Use Arrow Keys to control the snake.
